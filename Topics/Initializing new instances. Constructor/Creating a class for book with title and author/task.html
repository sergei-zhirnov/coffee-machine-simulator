Creating a class for book with title and author

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Imagine you are creating a Book class in Java, each book has a title and an author, and we can get its information via a method. Now, you need to write code to help with this. Firstly, you will receive a single line with a book title and an author separated by a comma ('Book Title,Author'). Create a new instance of the Book class with this information. Then, provide a method to return the information (title and author) of this book as a single string in the following format: 'Book Title by Author'.
Sample Input:
Moby Dick,Herman Melville

Sample Output:
Moby Dick by Herman Melville


Sample Input:
1984,George Orwell

Sample Output:
1984 by George Orwell
