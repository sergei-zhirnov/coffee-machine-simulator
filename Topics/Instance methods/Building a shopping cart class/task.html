Building a shopping cart class

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
You need to create a class named 'Cart' for an online store. The class should have one instance variable, 'total', which should represent the total price of the items in the cart. The class should also contain one instance method, 'addToCart', which takes an integer, representing the price of an item, as a parameter, adds the price to the total and then returns the new total. Create this class and use it to process a series of items being added to a cart. The input will be a single string, representing a series of item prices separated by space. The output should be a single integer, representing the total price of the items in the cart after all the items have been processed.
Sample Input:
1 2 3

Sample Output:
6


Sample Input:
5 10 15

Sample Output:
30



Memory limit: 256 MB
Time limit: 5 seconds
